# agro-export-bitova-himia-nextjs

Редизайн на сайта **bitova-himia.com** с изцяло Next.js архитектура и локални данни.

## Основни идеи
- Без runtime зависимост към Strapi.
- Изчистен, модерен и мобилно-адаптивен интерфейс.
- Ясна секция за набиране на дистрибутори в България.
- Минимални външни библиотеки.

## Стартиране
```bash
npm install
npm run dev
```

## Синхронизиране на данни от Strapi (по избор)
Скриптът изтегля съдържание от текущия Strapi и генерира `data/site-content.json`:

```bash
npm run sync:strapi
```

> Ако Strapi не е достъпен от средата ви, приложението работи с наличните локални данни.
